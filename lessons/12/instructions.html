This is exactly the same as setting as the previous lesson. We're merely replacing the three languages by shapes. So now we're modeling the probability that a triangle would be solid, striped, or hollow: that is, p(solid | triangle), p(striped | triangle), p(hollow | triangle). We will use this setup for the next several lessons.

Unlike the joint model of lesson 6, the conditional model does not try to predict the probability of triangles. The total expected number of triangles is fixed (at Ntriangle = 15), a number that is taken directly from the training data. The model only tries to predict the fill pattern given that it has a triangle.
